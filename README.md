# I-Y-F-Capsone-Project
This is a repository which includes my first project in the AI and Data Science Class of 2026
The project is about a school managemennt app that would solve the problem of miscommunication and help in passing important information to the school staff which includes teachers,non-teaching staff,students and their parents
The tech stack includes react which supports the Front-end for a fast and mobile design,Back-end will be supported by python for faster processing and AI-ready processing assistance the dDatabase will be supported by Supabase for file storage and user authentication the AI Engine will contain Hugging Face providing various models.
AI Educational Advisor:Personalized advice on misinterpretation of information,summarizing complex information and guiding the student on how to improve their grades and providing assistance on their assignment.
Marketplace:Specific schools that are interested in using the app for their benefit and for communicating effectively to their parents and other peaople of interest.
Personalized Onboarding:Educative content based on the curriculum provided by the school for effective learning.
Installation/Setup:
1.Python 3.10+ installed.
A Supabase account.
2.Backend Setup Open your terminal and navigate to the backend directory:cd backend
Create a virtual environment:python-m venv venv
Activate the environment:
Windows:venv\Scripts\activate
Install dependencies:pip install -requirements.txt
3.Frontend Setup Navigate to the frontend directory:cd frontend
Install dependencies:npm install
Educational Variables Create a .env file in the root of the relevant folders.Do not commit these files to Github.
frontend/.env
SUPABASE_URL=your project_url
SUPABASE_SERVICE_ROLE_KEY=Your_hugging face_key
Start Frontend:cd frontend & & npm run dev
